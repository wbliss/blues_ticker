# blues_ticker

Download and install:

python 3.4

LAMP

```
pip install flask
pip install flask_sqlalchemy
pip install pytz
pip install request
pip install simplejson
pip install pymysql
```

create a config.py file with variables: 

secret_key
sql_username
sql_password
fav_team_id (19 for blues)
localtz (For example 'Europe/Berlin', 'America/Chicago', etc.)

run setup.py
run controller.py
